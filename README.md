# Zomato Data-Analysis

# Project Overview
        This project performs in-depth Exploratory Data Analysis(EDA) on the Zomato restaurant dataset using Python.The goal is to analyze customer dining habits,restaurant ratings,and cost trends.
# Key features
      1) Data cleaning: Handled missing values and outliers to ensure dada integrity.
      2) EDA: Analyzed the relationship between 'Online Ordering' and 'Restaurant Rating'
      3) Visualization: Created Charts using Matplotlib and Seaborn to show restaurant density.
      4) Price Prediction: Basic model to predict the"Approx cost for two people".
# Technologies Used
      1) Language: Python.
      2) Libraries: Pandas,NumPy,Matplotlib,Seaborn.
      3) Tool: Jupyter Notebook.
