# Zomato Data-Analysis

(dataset sourced from kaggle)
## Project Overview
 This project performs in-depth Exploratory Data Analysis(EDA) on the Zomato restaurant dataset using Python.The goal is to analyze customer dining habits,restaurant ratings,and cost trends.
## Key features
   - Data cleaning: Handled missing values and outliers to ensure data integrity.
   - EDA: Analyzed the relationship between 'Online Ordering' and 'Restaurant Rating'
   - Visualization: Created Charts using Matplotlib and Seaborn to show restaurant density.
   - Price Prediction: Basic model to predict the"Approx cost for two people".
## Technologies Used
   - Language: Python.
   - Libraries: Pandas,NumPy,Matplotlib,Seaborn.
   -  Tool: Jupyter Notebook.
